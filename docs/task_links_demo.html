Pattern Recognition

Task IDs following the pattern #task_<id> are automatically converted to clickable links.

Example Messages
Assistant Message

I've created #task_abc123 to handle your request. You can monitor progress in the dashboard.

Assistant Message

Task #task_fix_bug_20241027 has completed successfully. The changes are now live.

User Message

What's the status of #task_abc123?

Mixed Content

Check #task_deploy_20241027 for deployment logs. Also see documentation for more info.

Features
  • Automatic detection of task ID patterns
  • Distinct orange styling for easy identification
  • Opens monitoring dashboard in new tab
  • Hover effects provide visual feedback
  • Works in both user and assistant messages
  • Compatible with markdown formatting
Supported Patterns

#task_abc123 - Alphanumeric ID
#task_20241027_deploy - Date-prefixed ID
#task_fix_bug_auth - Descriptive ID
#task_TEST_ENV_123 - Uppercase allowed
